# Console Calculator

A simple Java-based console calculator that allows the user to perform basic arithmetic operations on two numbers.  

## Features
- Addition, Subtraction, Multiplication, and Division
- User selects the operation to perform
- Console-based interaction

## Usage
1. Run the program in IntelliJ or any Java IDE.
2. Enter your first number when prompted.
3. Choose the desired operation (add, subtract, multiply, divide).
4. Enter your second number when prompted.
5. See the result printed in the console.

## Notes
- This is a simple educational project.
- No GUI, console-based only.
